# 🎮 Gaming Website 

A modern **animated gaming landing page** built using **React, TailwindCSS, and GSAP**.  
This project focuses on **smooth animations, immersive visuals, and responsive design** to create a cinematic gaming website experience.

The website showcases different sections including a **hero video banner, animated content cards, story section, and call-to-action areas**, all enhanced with **GSAP scroll animations and transitions**.

---

## 🚀 Features

- ⚡ Built with **React**
- 🎨 Styled using **Tailwind CSS**
- 🎬 Smooth animations using **GSAP**
- 📱 Fully **Responsive Design** (Desktop, Tablet, Mobile)
- 🎥 Animated **Hero Section with video**
- 🧩 Interactive **Content Cards**
- 📖 Storytelling section with animated layout
- 🖱 Scroll-triggered animations
- 🎯 Clean and modern UI

## ⚙️ Installation & Setup

Clone the repository and install dependencies.

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/MuhammadWaris-55/Gaming-Website.git
```

### 2️⃣ Install Dependencies

```bash
npm i
```

### 3️⃣ Run the Development Server

```bash
npm run dev
```

The project will start on:

```
http://localhost:5173
```
